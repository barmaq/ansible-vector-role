Vector
=========

Эта роль установит Vector на debian\ubuntu


Переменные
--------------

| Переменная  | Назначение  |
|:---|:---|
| `vector_version` | версия `Vector` |


Пример Playbook
----------------


    - hosts: servers
      roles:
         - { role: ansible-vector }
